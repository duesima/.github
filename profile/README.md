# Duesima
Engineering systems integration, modeling, and analysis.
Duesima is an engineering software project focused on integrated models of buildings and physical systems. Current development includes building systems, HVAC loads and ventilation, thermodynamics, psychrometrics, geometry, and unit-aware computation.
The software is in active early development.
